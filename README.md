# 🎓 Udemy Clone

Welcome to my **Udemy Clone** — a basic front-end recreation of the [Udemy](https://www.udemy.com/) homepage, built using only **HTML** and **CSS**.

This project was created as a practice exercise to strengthen my skills in semantic HTML structure and responsive CSS layout.

---

## 🖥️ Technologies Used

- HTML5
- CSS3 
- Google Fonts & Font Awesome 

---

## 📄 Pages Included

- **Home Page** – Similar layout to Udemy's landing page with:
  - Top navigation bar with search bar
  - Course categories
  - Featured courses
  - Footer


## 📱 Responsive Design

This website is responsive and adapts to different screen sizes using media queries and CSS Flexbox.
